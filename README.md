# dao-skill ｜ 道．skill

> 道可道，非常道；名可名，非常名。

把《道德经》五千言，**蒸馏成一个会说话的老子**——你带着生活里的困惑来，他以第一人称「我」，用自己的世界观，平静地把那条本来就在你脚下的道，指给你看。

属于 [awesome-persona-skills](https://github.com/hewei5210/awesome-persona-skills) 「传统文化与哲学」一类的人格蒸馏 Skill。

---

## 这是什么

这是一个 **Agent Skill**（Claude / Cursor 等可加载的技能）。加载后，AI 不再以助手的口吻答你，而是**化身老子本人**，依据《道德经》通行本（王弼本）的义理，为你的现实困扰做开解。

- **人设**：老子第一人称「我」，简朴、慈悲、不说教，善用水、婴儿、山谷、未雕之木等自然意象，惯用「正言若反」。
- **底本**：通行本（魏·王弼注本），八十一章；凡引原文力求一字不差并可标章次。
- **定位**：人生处世开解——焦虑内卷、得失进退、感情放不下、欲望与知足、迷茫与决策、为人与处下。

## 它能为你做什么

| 你的困扰 | 老子会怎么点你 |
|---|---|
| 内卷、争不过别人 | 「夫唯不争，故天下莫能与之争。」学水之处下，自胜者强、知足者富。 |
| 正得意、怕守不住 | 「持而盈之，不如其已；功遂身退，天之道。」 |
| 感情/往事放不下 | 「致虚极，守静笃。」万物各复归其根，不追不赶。 |
| 被「以后」压得喘不过气 | 「知足不辱，知止不殆。」名与身孰亲？身与货孰多？ |
| 迷茫、不知追求什么 | 「见素抱朴，少私寡欲。」活成你本来的样子。 |
| 道理都懂却做不到 | 「千里之行，始于足下。」道不是用来懂的，是用来行的。 |

## 安装与使用

将本仓库作为一个 skill 目录放入你的 Agent 技能目录，例如：

```bash
# Claude Code（个人技能目录）
git clone https://github.com/hewei5210/dao-skill ~/.claude/skills/dao-de-jing

# 或放入你的 Cursor / Agent 所约定的 skills 目录
```

加载后，直接把困惑说给它即可，例如：

> 「我天天加班还是升不上去，很不甘心。」
>
> 「我升职了，但总有点不踏实。」
>
> 「一段感情过去几年了，还是放不下。」

它会以老子的口吻，引《道德经》原文为你开解。

## 文件结构

| 文件 | 作用 |
|---|---|
| `SKILL.md` | 主入口：YAML 头 + 义理体系（Part A）+ 说法角色（Part B），自包含、加载即生效 |
| `doctrine.md` | 义理体系全文：道的世界观（核心义理、重要概念、名句库、应用心法、常见误区） |
| `persona.md` | 说法角色全文：老子第一人称的戒律、身份、风格、应机、边界 |
| `meta.json` | 元数据：底本、人设、核心思想、来源 |
| `README.md` | 本文件 |

## 引用与底本

- 全部经文以**通行本（王弼注本）**为准。
- AI 引用原文时力求一字不差并标章次；不确定处会注明「大意是……」，不杜撰。
- 注疏背景参考王弼（玄学）、河上公（养生）及现代生活化解读，但引文一律回到王弼本。

## 一点说明（边界）

老子只治心、不治身，更不卜吉凶。这个 skill **不算命、不看相、不断祸福**，也不替代医疗与法律。身有疾、事涉法，请去寻对的人。它能做的，是陪你把心静下来，看清困惑本来的样子。
